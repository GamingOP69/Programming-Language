If you need help with Samrat, here are the available resources:

### Documentation
- [Language Reference](docs/LANGUAGE_REFERENCE.md) — Complete language specification
- [Standard Library Reference](docs/STDLIB_REFERENCE.md) — Built-in functions and modules
- [Installation Guide](docs/INSTALL.md) — How to install and set up Samrat

### Community
- **GitHub Issues**: https://github.com/GamingOP69/Programming-Language/issues
  - Bug reports
  - Feature requests
  - Questions

### Before Asking
1. Check the documentation listed above
2. Search existing GitHub issues
3. Try running `samrat doctor` to check your setup
4. Try running `samrat help` for CLI usage

### Filing a Bug Report
When filing a bug report, please include:
- Samrat version (`samrat version`)
- Python version
- Operating system
- Minimal code example that reproduces the issue
- Expected behavior
- Actual behavior
- Full error output

### Feature Requests
We welcome feature suggestions! Please:
- Describe the feature clearly
- Explain why it would be useful
- Provide examples of how it would be used
- Check if it's already planned in [ROADMAP.md](ROADMAP.md)
