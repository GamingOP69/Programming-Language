# Samrat Roadmap

## Version 1.0.0 — Current Release ✅

- [x] Lexer with full token support
- [x] Recursive-descent parser with precedence climbing
- [x] Abstract Syntax Tree (AST)
- [x] Semantic analyzer with type checking
- [x] Tree-walking interpreter
- [x] Bytecode compiler
- [x] Stack-based virtual machine
- [x] REPL (interactive mode)
- [x] CLI with 14 commands
- [x] 18 example programs
- [x] 491+ tests at 100% pass rate
- [x] Comprehensive documentation
- [x] MIT License
- [x] CI/CD with GitHub Actions

## Version 1.1.0 — Performance & Polish

- [ ] Compiler optimizations (constant folding, DCE, peephole)
- [ ] JIT compilation (basic)
- [ ] Performance benchmarks
- [ ] Improved error messages with source context
- [ ] Source maps for debugging
- [ ] Formatter improvements
- [ ] Linter

## Version 1.2.0 — Standard Library

- [ ] Complete math module (trig, stats, random)
- [ ] String module (regex, formatting)
- [ ] File I/O module (read/write/append)
- [ ] JSON module (parse/stringify)
- [ ] HTTP client module
- [ ] DateTime module
- [ ] Collections module (deque, counter, etc.)
- [ ] OS module (env vars, path, process)

## Version 2.0.0 — Major Features

- [ ] Native code generation (LLVM backend)
- [ ] Type system enhancements (generics, unions, interfaces)
- [ ] Pattern matching (match/case)
- [ ] Async/await support
- [ ] Package manager
- [ ] Language server protocol (LSP) support
- [ ] Debugger
- [ ] WebAssembly target

## Version 3.0.0 — Ecosystem

- [ ] Package registry
- [ ] IDE plugins (VS Code, JetBrains)
- [ ] Documentation generator
- [ ] Testing framework
- [ ] Build system
- [ ] Cross-compilation
- [ ] Self-hosting compiler
